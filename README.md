# npm-downloads

> Get the downloads count from a given npm package

## Install

```sh
$ npm install npm-downloads -g
```

## Usage

```sh
Usage: npm-downloads [OPTIONS]

  Get the downloads count from a given npm package

Example:
  $ npm-downloads .

Options:
  -v --version              Display current software version
  -h --help                 Display help and usage details
```
